# 💰 Smart Spend AI

A smart, AI-powered personal expense tracker web app built for hackathon submission.

## 🔗 Live Demo
[https://shesanths6-coder.github.io/SmartSpend-ai/](https://shesanths6-coder.github.io/SmartSpend-ai/)

## ✨ Features
- 🔐 Login & Signup
- 📊 Dashboard with real-time stat cards
- 💸 Expense Add / Track / Delete
- 🥧 Pie Chart (category-wise spending)
- 🤖 Smart AI Insights
- ⚠️ Budget Warning Alerts
- 🎤 Voice Assistant (speech-to-expense)
- 📷 Bill Scanner (OCR)
- 📍 Location-based Expense Tracking
- 📅 Monthly History
- 🔍 Category Filter
- 🌓 Dark/Light Mode
- ⬇️ Export to CSV
- 📱 Mobile Responsive

## 🛠️ Built With
- HTML, CSS, JavaScript
- Chart.js (data visualization)
- Tesseract.js (OCR bill scanning)
- Web Speech API (voice input)
- Geolocation API + OpenStreetMap (location tracking)
- LocalStorage (data persistence)
